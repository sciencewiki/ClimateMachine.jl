compute_buoyancy_flux(param_set, shf, lhf, T_b, q, α_0)

Computes buoyancy flux given

  • shf sensible heat flux
  • lhf latent heat flux
  • T_b surface boundary temperature
  • q phase partition (see PhasePartition)
  • α_0 specific volume

CLIMA.SurfaceFluxes.Nishizawa2018.compute_friction_velocity — Function
compute_friction_velocity(param_set, u_ave, θ, flux, Δz, z_0, a, Ψ_m_tol, tol_abs, iter_max)

Computes friction velocity, in Eq. 12 in, by solving the non-linear equation:

u_ave = ustar/κ * ( ln(Δz/z_0) - Ψ_m(Δz/L) + z_0/Δz * Ψ_m(z_0/L) + R_z0 [ψ_m(z_0/L) - 1] )

where L is a non-linear function of ustar (see monin_obukhov_len).
